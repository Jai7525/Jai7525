# 🤖 DevGuide AI

### AI-Powered Codebase Understanding Platform

**AI Engineering • Developer Productivity • Repository Intelligence**

> Transform software repositories into searchable knowledge systems using semantic retrieval, Retrieval-Augmented Generation, and contextual AI explanations.

<br>

## 📖 Overview

DevGuide AI helps developers explore unfamiliar codebases without manually navigating hundreds of files.

It scans uploaded repositories, processes source code, generates embeddings, builds a FAISS vector index, and allows developers to ask repository-specific questions through a source-grounded RAG pipeline.

# 🛡️ InboxIQ AI

### AI-Powered Email Security Platform

**AI Security • Threat Intelligence • Explainable Analysis**

> Analyze Gmail messages for phishing indicators, suspicious URLs, spoofed domains, and email-based threats using AI-assisted security intelligence.

<br>

## 📖 Overview

InboxIQ AI is an intelligent email-security platform that provides explainable phishing analysis instead of only classifying messages as safe or unsafe.

It combines Gmail API, Google OAuth, Google Safe Browsing, domain intelligence, risk scoring, and Retrieval-Augmented Generation to generate contextual security findings and actionable threat insights.
