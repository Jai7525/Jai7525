# DevGuide AI
### AI-Powered Codebase Understanding Platform

</summary>

<br>

## Overview

DevGuide AI is an AI-powered software engineering platform that helps developers understand large repositories through semantic search, Retrieval-Augmented Generation (RAG), and AI-generated documentation.

Instead of manually exploring hundreds of source files, developers upload a repository and interact with the entire codebase using natural language.

---

## Engineering Problem

Understanding unfamiliar repositories is slow and requires manually reading numerous files before making changes.

---

## Solution

DevGuide AI automatically:

- Scans repositories
- Parses source code
- Chunks files intelligently
- Creates embeddings
- Stores vectors in FAISS
- Retrieves relevant context
- Generates grounded AI explanations using Gemini AI

---

## Technology Stack

|--------|------------|
| Frontend | React + Tailwind CSS |
| Backend | FastAPI |
| Database | PostgreSQL |
| AI | Gemini AI |

| Deployment | Render |
